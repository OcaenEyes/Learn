<!-- 常见的几个报错类型 -->
NameError                尝试访问一个没有申明的变量  <br>
ZeroDivisionError         除数为0 <br>
SyntaxError				  语法错误 <br>
IndexError				  索引超出序列范围 <br>
KeyError				  请求一个不存在的字典关键字. <br> 
IOError					  输入输出错误（比如要读取的文件不存在） <br>
AttributeError			  尝试访问未知的对象属性    <br>
ValueError				  传给函数的参数类型不正确，比如给int()函数传入字符串类型   <br>
